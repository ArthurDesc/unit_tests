
## Installation

1. Installer les dépendances PHP :
   ```bash
   composer install
   ```
2. Installer les dépendances JavaScript :
   ```bash
   npm install
   ```

## Lancer les Tests

### Tests PHP (PHPUnit)
```bash
composer test
```

### Tests JavaScript (Jest)
```bash
npm test
```
